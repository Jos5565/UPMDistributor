### UPM Distibutor
> UPM Distibutor is publish Tool for UPM(Unity Package Manager)

### How to use
> 1. Download Unity Package from [Release Page](https://github.com/Jos5565/UMPDistributor/releases/tag/UPMDistributor)
> 2. And Install to Unity Project.
> 3. Click "UMP Publish" -> "UMP Distributot"
